# 🧠 Maya Brainstorm

**Three AI agents debate any topic. You steer the discussion.**

Visionary · Pragmatist · Skeptic — three perspectives, one question.
Each round they read each other's arguments and go deeper.
You can jump in between rounds and redirect the debate.

## How it works

1. Enter any topic
2. Three AI agents discuss it in parallel
3. A moderator synthesizes key ideas after each round
4. You add your comment — agents prioritize it in the next round
5. Up to 5 rounds total
